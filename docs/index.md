# MultiTrust

A trust framework SDK for multi-agent AI systems, grounded in Subjective
Logic. Trust is modeled as **opinions** (belief, disbelief, uncertainty)
rather than raw scores, with operators to fuse, discount, and decay trust
over time.

This site is the in-progress documentation scaffold for MultiTrust. The
full developer guide and cookbook land in later phases; for now, the two
existing references are:

- [Developer design](dev_design.md) — architecture, data flow, and the
  reasoning behind the core types.
- [Experimental API reference](exp_api.md) — current public surface,
  pending the Phase 0 inventory and the Phase 5 freeze.

For installation and quickstart, see the project
[README](https://github.com/nobelk/multitrust#readme).
